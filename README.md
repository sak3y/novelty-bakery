# Novelty Bakery Website

A small brochure-style website for a Bakery, built with React, React Router and Tailwind CSS. The site showcases everyday products, shares the bakery story, and makes it easy for visitors to find and contact the shop.

## Tech stack

- React + TypeScript  
- React Router for page navigation  
- Tailwind CSS for styling  
- Vite as the build tool


## Running the project
1. `Git clone https://github.com/sak3y/novelty-bakery.git`
2. `npm i`
3. `npm run dev`
